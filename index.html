Portfolio Projects:

  • Blog Form n Feed
    • A full-stack project using Flask, AlpineJS, and Picnic CSS.
    • Create a blog via the form and view it individually as a post, or collectively with other blogs in the feed.
  • Bop them Boids
    • A Godot game adaptation of the boid algorithms developed by Craig Reynolds in 1986, demonstrating flocking behavior.
    • Catch as many jellyfish as you can! Your fish follows your cursor and the boids avoid the fish.
    • My record is 32 boids bopped.
  • SEO Chrome Extension
    • A Typescript chrome extension project in the form of a light SEO tool.
    • Features include:
      • Inject a persistent SEO checklist as a collapsible sidebar.
      • View SEO data conveniently via the extension popup.
  • Blog Pixie
    • A file exchanger for cropping, resizing, and renaming an image.
    • Made with React.
  • Jackbox Games Clone
    • A full-stack project to create a Jackbox Games clone, requiring three parts:
      1. Frontend
        • A web page for interacting with the game and your character.
      2. Backend
        • A FastAPI server for managing the state of all sessions and their games, players, scores, etc
        • Communicates data between the frontend and the game engine using the Websocket protocol.
      3. Game Engine
        • A game engine, Godot, for handling game logic and graphics rendering.
